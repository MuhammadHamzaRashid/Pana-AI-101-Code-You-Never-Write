The objective of this project is to calculate my semester grade according to my teacher's grading policy.

### Teacher's Grading Policy

- Assignments: 30 marks (lowest assignment score is dropped)
- Mid-Term Exam: 120 marks
- Final Exam: 150 marks
- Total: 300 marks

The program calculates:
- Assignment marks after dropping the lowest score
- Total semester marks
- Percentage
- Final letter grade

## AI Tool Used

I used **ChatGPT** to help write the Python program and explain the grading calculations based on the teacher's grading policy.

## Verification

To verify the result, I manually calculated the assignment marks by dropping the lowest assignment score and scaling the remaining marks to the 30-mark assignment category. I then added the mid-term and final exam marks to obtain the total score and percentage. The manual calculation matched the output produced by the Python program, confirming that the result is correct.
